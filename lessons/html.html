Lecture 01 · Foundations

What is HTML?

Beginner ~40 min No prerequisites

Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web.

hello.html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Page</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>Welcome to <strong>HTML</strong>.</p>
</body>
</html>

Document Structure

The HTML Boilerplate

  <meta charset="UTF-8">

</head>
<body>
  <!-- Content here -->
</body>
</html>

Lecture 04 · Foundations

Links & Navigation

Beginner ~45 min Requires: Lecture 03

The Anchor Tag

<!-- External link -->
<a href="https://example.com">Visit</a>

<!-- New tab -->
<a href="..." target="_blank" rel="noopener">Open New Tab</a>

<!-- Email -->
<a href="mailto:email@example.com">Email</a>

<!-- Anchor -->
<a href="#section">Jump</a>
<h2 id="section">Section</h2>
Lecture 05 · Foundations

Images & Media

Beginner ~50 min Requires: Lecture 04

Adding Images

<!-- Basic image -->
<img src="photo.jpg" alt="Description">

<!-- With dimensions -->
<img src="photo.jpg" alt="Description" 
     width="600" height="400">

<!-- Figure with caption -->
<figure>
  <img src="chart.png" alt="Chart">
  <figcaption>Figure 1: Sales</figcaption>
</figure>
